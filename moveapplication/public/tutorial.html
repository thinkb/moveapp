Começar

Seja-bem vindo ao Move!

Aqui você se diverte enquanto pratica exercícios físicos.

Para conseguir pontos e subir de nível

Você deve aceitar e cumprir desafios.

Os desafios são organizados por treinador.

Cada um tem sua especialidade.

Cada desafio realizado você ganha um sticker para colecionar.

Quando uma página esta completa você recebe um código para trocar por um sticker para sua pulseira .

Aperte a pulseira sempre que quiser atualizar os dados.
